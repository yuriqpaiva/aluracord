Simple discord chat using Next.js, SkynexUI and Supabase as database CMS.

## How to Install Dependencies?

Using NPM

```
npm install
```

Using Yarn:

```
yarn install
```

## How to run it?

Using NPM

```
npm run dev
```

Using Yarn:

```
yarn dev
```

## How to build for production?

Using NPM

```
npm run build
```

Using Yarn:

```
yarn build
```
